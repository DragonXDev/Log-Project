What is a log?
A log is a quantity representing the power to which a fixed number, or the base, must be raised to produce a given number.

Product Property
The sum of two logarithms with the same base is equal to the logarithm of the product of the two arguments.
log a + log b = log ab
log 4 + log 3 = log 12
log 5 + log 6 = log 30
log 9 + log 2 = log 18
Quotient Property
The difference of two logarithms with the same base is equal to the logarithm of the quotient of the two arguments.
log a - log b = log a/b
log 4 - log 3 = log 4/3
log 5 - log 6 = log 5/6
log 9 - log 2 = log 9/2
Power Property
If the argument inside a logairthm is raised to an exponent, then the exponent can be taken out as a factor multiplying the logarithm.
log a^b = b * log a
log 4^3 = 3 * log 4
log 5^2 = 2 * log 5
log 9^4 = 4 * log 9
Inverse Property
Due to the fact that exponentiation is the inverse of logarithms, a log of it's own base cancels out, and a base to a log of that base cancels out as well, leaving the other elements behind.
b ^[ log[base b](x) ] = x, log[base b] b^x = x
log [base 4] (4^3) = 3
log [base 5] (5^2) = 2
log [base 18] (18^17) = 17
Change of Base Property
To change the base of a logarithmic equation, it is equivalent to the log with the new base of the original argument divded by the log with the new base of the old logarithmic base.
log [base b] c = ( log [base a] c ) / ( log [base a] b )
log [base 4] 3 = (log 3)/(log 4)
log [base 5] 2 = (log 2)/(log 5)
log [base 872] 983 = (log [base 812] 983) / (log [base 812] 872)
